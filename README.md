# 【概要】
[Legmin/Webcam-python](https://github.com/Legmin/Webcam-python.git)のデバックルームです。
opencvやpydriveを使っているプロジェクトです。

# 【説明】
## folder/hojo_list.txt
```
HOJO_A
HOJO_B
HOJO_C
:
:
```
カメラを設置する圃場名を**半角、改行区切り**で入力します。
**圃場を追加/消去**する場合、ここを書き換えます。
## picture/
```
HOJO_A_20xxxxxx_xxxxxx.png
HOJO_A_20xxxxxx_xxxxxx.png
:
:
```
撮影された写真はここに格納されます。
